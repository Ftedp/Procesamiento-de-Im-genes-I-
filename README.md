# 📷 Procesamiento Digital de Imágenes I

Repositorio de ejercicios y trabajos prácticos de la materia **Procesamiento Digital de Imágenes I** de la carrera **TUIA - UNR**. Implementaciones en Python utilizando OpenCV y NumPy.
Cada carpeta contiene los scripts correspondientes a cada unidad temática y trabajo práctico, organizados para facilitar su exploración y ejecución.

---

## 📚 Temas de la materia

| # | Unidad |

| 4 | Restauración de Imágenes |
| 5 | Procesamiento de Color |

## 🚀 Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/Ftedp/Procesamiento-de-Im-genes-I-.git
cd Procesamiento-de-Im-genes-I-
```

### 2. Crear el entorno virtual

```bash
python -m venv venv-Procesamiento_de_Imagenes_I
```

### 3. Activar el entorno virtual

**Windows (PowerShell):**
```powershell
venv-Procesamiento_de_Imagenes_I\Scripts\Activate.ps1
```

**Linux / macOS:**
```bash
source venv-Procesamiento_de_Imagenes_I/bin/activate
```

### 4. Instalar las dependencias

```bash
pip install opencv-python numpy matplotlib scipy
```

---

## 📦 Generar requirements.txt

Con el entorno virtual activado, podés exportar las dependencias con:

```bash
pip freeze > requirements.txt
```
